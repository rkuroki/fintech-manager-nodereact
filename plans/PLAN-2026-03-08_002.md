# Implementation Plan — 2026-03-08_002

## Overview
Gmail integration for the Communications section of the Customer Detail page. Analysts using corporate Gmail (Google Workspace) will see a **unified timeline** of all email correspondence (sent + received) with each customer, merged chronologically with existing manual communication records.

## Design Decisions
| Decision | Choice | Rationale |
|---|---|---|
| Auth method | **Per-user OAuth2** | Each analyst connects their own Gmail; no admin service account needed |
| UI layout | **Unified timeline** | Gmail emails + manual records in single chronological view |
| Data strategy | **Cache in DB + periodic sync** | Email metadata stored locally; on-demand sync via button |
| Gmail API | **Native fetch** | No `googleapis` npm package; fewer dependencies |
| Scope | **gmail.readonly** | Read-only; app never sends email on behalf of users |
